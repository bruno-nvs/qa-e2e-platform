# QA E2E Platform - Playwright

Projeto desenvolvido com foco em automação end-to-end utilizando Playwright, aplicando fluxo de trabalho baseado em versionamento profissional, validação contínua e integração com GitHub Actions.

O objetivo deste projeto é praticar automação moderna de testes, execução multiambiente e fluxo corporativo utilizando branches, pull requests e pipeline automatizada.

---

## Objetivo

Validar fluxos web automatizados utilizando Playwright e boas práticas de desenvolvimento colaborativo.

O projeto também foi utilizado para prática de:

- Estruturação de testes E2E
- Versionamento com Git
- Estratégia de branches
- Pull Requests
- Integração contínua (CI)
- Execução automatizada via GitHub Actions

---

## Tecnologias Utilizadas

- Node.js
- Playwright
- Git
- GitHub
- GitHub Actions

---
